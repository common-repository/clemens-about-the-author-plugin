Adds an about-the-author box after every post and page. Includes a gravatar-image and a biography from the author.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin-directory  `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You're done

== Frequently Asked Questions ==

= The stylesheet doesn't work! =

Make sure you have uploaded the whole PLUGIN DIRECTORY to /wp-content/plugins/ because otherwise the stylesheet
won't get included correctly.
